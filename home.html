What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for HTML and XML documents that represents the document as a tree of nodes. It provides a way for JavaScript code to interact with and manipulate the document, and is used by web browsers and search engines to render and index web pages.

How do you select element from the DOM?

There are several ways to select elements from the DOM, depending on the specific use case and the document's structure. Use document.querySelector() or document.querySelectorAll() instead of document.getElementById() or document.getElementsByTagName() for more flexibility.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique used in JavaScript to handle events on a group of elements by attaching a single event listener to a parent element. This approach takes advantage of the fact that events "bubble up" the DOM tree, meaning that an event triggered on a child element will also trigger on its parent elements.

How do you manipulate an element's attributes and styles useing the DOM?
